Introduction
============

**AdminLTEQueimadas** -- is a fully responsive admin template. Based on **[Bootstrap 3](https://github.com/twbs/bootstrap)** framework. Highly customizable and easy to use. Fits many screen resolutions from small mobile devices to large desktops. This template was adapted from AdminLTE (https://github.com/almasaeed2010/AdminLTE) to be used in the INPE Queimadas department (http://www.inpe.br/queimadas/) systems.

Installation
------------

**Github**

- Clone the repository
```
git clone https://github.com/YOUR_USERNAME/AdminLTE.git
```

**Bower**

```
bower install admin-lte
```

**Composer**

```
composer require "almasaeed2010/adminlte=~2.0"
```

Browser Support
---------------
- IE 9+
- Firefox (latest)
- Chrome (latest)
- Safari (latest)
- Opera (latest)
